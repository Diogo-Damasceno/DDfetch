# 🚀 DDfetch

A lightweight and customizable system information tool written in Bash, inspired by tools like fastfetch and neofetch.

DDfetch displays system information alongside a custom ASCII art in a clean and minimal layout — perfect for terminal aesthetics and r/unixporn setups.

---

## ✨ Features

- 🧠 Written in pure Bash (no dependencies)
- 🎨 Custom ASCII art support
- ⚡ Fast and lightweight
- 🖥 Displays system info:
  - OS
  - Kernel
  - Packages
  - Shell
  - Window Manager
  - CPU
  - RAM
- 🔧 Easy to modify and extend
